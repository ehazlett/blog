Shipyard: Applications

Tue, Jul 9, 2013 The next major feature for Shipyard is “applications”. These are a collection of containers that are accessible by a domain name. Shipyard uses Hipache for the frontend load balancer. Using Shipyard, you can launch a set of containers that expose a backend port. You can then create an “application” which configures Hipache and routes traffic to all containers in that app on the container backend port. Configuration is automatic and you can have multiple containers across multiple hosts and Hipache will balance and direct traffic to them. read more...
